# Homebrew Tap for Pipe Pile Optimizer

This is a [Homebrew](https://brew.sh/) tap for [Pipe Pile Optimizer](https://github.com/kavanaghpatrick/pipe-optimizer).

## Installation

```bash
brew install kavanaghpatrick/pipe-optimizer/pipe-optimizer
```

Or, if you prefer to tap first:

```bash
brew tap kavanaghpatrick/pipe-optimizer
brew install --cask pipe-optimizer
```

## What is Pipe Optimizer?

A standalone desktop application that finds the mathematically optimal way to combine pipe segments into piles of a target length, minimizing waste.

- Uses Integer Linear Programming (ILP) for provably optimal solutions
- Handles 750+ pipes in minutes
- Cross-platform: Mac and Windows

## Updating

```bash
brew upgrade pipe-optimizer
```

## Uninstalling

```bash
brew uninstall pipe-optimizer
```
